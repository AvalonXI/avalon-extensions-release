# avalonbeta

Ashita v4 addon: floating ImGui panel of AvalonXI dev / beta-test `!` commands.

## What it does

- A lean, draggable panel that mirrors the in-game `!dev` index.
- Dropdowns drive `!loadouts` / `!prema` for instant grants; `!missions` / `!quests` / `!nms` / `!hnms` take inline params; the rest are plain buttons.
- Panel visibility, position, and size persist per character.

## Load

`/addon load avalonbeta`

## Commands

- `/beta` - toggle the panel
- `/beta show` | `hide` | `toggle`
- `/beta lock [on|off]` - lock/unlock the window (fixed overlay)
- `/beta reload` | `reset`
- `/beta help`
