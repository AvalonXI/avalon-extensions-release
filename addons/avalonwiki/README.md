# avalonwiki

Ashita v4 addon for opening the AvalonXI Miraheze wiki from inside the game.

## What it does

- `/wiki` opens the wiki main page.
- `/wiki <query>` opens the wiki search/go for that page or term.

## Load

`/addon load avalonwiki`

## Commands

- `/wiki`
- `/wiki <page name or query>`
